# 핏자워크라운지 어드민 배포 가이드

## 준비물
- GitHub 계정 (무료)
- Supabase 계정 (무료) → https://supabase.com
- Vercel 계정 (무료) → https://vercel.com

---

## STEP 1. Supabase 데이터베이스 설정

1. https://supabase.com 접속 → 구글 계정으로 가입/로그인
2. **New Project** 클릭
   - 이름: `fitzawork-admin`
   - Database Password: 기억할 수 있는 비밀번호 설정
   - Region: **Northeast Asia (Tokyo)** 선택 → Create Project
3. 프로젝트 생성 후 왼쪽 메뉴 **SQL Editor** 클릭
4. `supabase_schema.sql` 파일 내용 전체 복사 → 붙여넣기 → **Run** 클릭
5. 왼쪽 메뉴 **Settings → API** 클릭
   - `Project URL` 복사해두기
   - `anon public` 키 복사해두기

---

## STEP 2. GitHub에 코드 올리기

1. https://github.com 접속 → 로그인
2. 오른쪽 위 **+** → **New repository**
   - Repository name: `fitzawork-admin`
   - Private 선택 → **Create repository**
3. 컴퓨터에서 터미널(또는 Terminal.app) 열기
4. 아래 명령어 순서대로 입력:

```bash
cd ~/Documents/Claude/Projects/핏자워크라운지/admin-web
git init
git add .
git commit -m "first commit"
git branch -M main
git remote add origin https://github.com/[내깃헙아이디]/fitzawork-admin.git
git push -u origin main
```

---

## STEP 3. Vercel 배포

1. https://vercel.com 접속 → 구글 계정으로 가입/로그인
2. **Add New → Project** 클릭
3. GitHub 연결 → `fitzawork-admin` 저장소 선택 → **Import**
4. **Environment Variables** 섹션에서 아래 두 가지 추가:

   | Key | Value |
   |-----|-------|
   | `NEXT_PUBLIC_SUPABASE_URL` | STEP 1에서 복사한 Project URL |
   | `NEXT_PUBLIC_SUPABASE_ANON_KEY` | STEP 1에서 복사한 anon public 키 |

5. **Deploy** 클릭 → 1-2분 후 완료
6. 생성된 링크 (예: `fitzawork-admin.vercel.app`) 로 접속!
